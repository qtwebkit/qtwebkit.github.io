Qt WebKit QML Types

The QML types can be imported into your applciation using the following import statement in your .qml file:

import QtQuick 2.0
import QtWebKit 3.0

WebLoadRequest

A utility class for the WebView::loadingChanged signal

WebView

A WebView renders web content within a QML application
